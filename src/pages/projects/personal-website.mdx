<span class="text-lg font-semibold">What is This?</span>

This is my **personal portfolio website**, built to showcase my projects, blog posts, and background.
Instead of using a modern minimalist design, I decided to create a **retro Windows 95 aesthetic**,
mixing nostalgic UI with clean, accessible code.

<br />
<span class="text-lg font-semibold">Key Features</span>

<ul class="list-disc ml-6">
  <li>
    **Retro UI:** Styled with Windows 95-inspired components and pixel fonts.
  </li>
  <li>
    **Responsive design:** Works seamlessly on both desktop and mobile while
    keeping the retro look.
  </li>
  <li>
    **Projects & blog:** Each project has its own “window” with details, and the
    blog highlights my learnings.
  </li>
  <li>
    **Resume & contact:** Easy access to connect with me or download my CV.
  </li>
</ul>

<br />
<span class="text-lg font-semibold">Technical Highlights</span>

I treated this as more than just a design experiment - it’s also a chance to practice modern frontend techniques:

<ul class="list-disc ml-6">
  <li>
    **Framework:** Astro for fast static generation and simple content
    management.
  </li>
  <li>
    **Styling:** Tailwind CSS with custom themes (Win95 palette, pixel fonts).
  </li>
  <li>
    **Performance:** Optimized assets, image strategies, and responsive
    typography.
  </li>
  <li>
    **Accessibility:** Semantic HTML and ARIA roles so screen readers handle
    decorative elements properly.
  </li>
  <li>
    **Deployment:** CI/CD pipeline with GitHub Actions for automatic builds and
    deploys.
  </li>
</ul>

<br />
<span class="text-lg font-semibold">Tech Stack</span>

<ul class="list-disc ml-6">
  <li>**Frontend:** Astro, Tailwind CSS</li>
  <li>**Deployment:** Netlify</li>
  <li>
    **Design:** Custom fonts (IBM Plex Mono, IBM_VGA_8x16, Microsoft
    Sans-Serif), Windows-inspired UI
  </li>
</ul>

<br />
<span class="text-lg font-semibold">Why I Built This</span>

I wanted my portfolio to reflect both my **technical skills** and my **personality**.
Instead of a generic template, I built something memorable that combines my love for coding with a nostalgic retro style.
It’s not just a portfolio - it’s also a project in itself that demonstrates my **frontend, design, and deployment skills**.
