<span class="text-lg font-semibold">What is Lingput?</span>

Lingput is a **full-stack, production-grade web app** that helps learners acquire a new language through **short, AI-generated stories** tailored to their vocabulary.
Unlike flashcards, Lingput provides **natural comprehensible input** — stories, translations, audio, and smart word tracking — making the learning process immersive and effective.

<br />
<span class="text-lg font-semibold">Technical Highlights</span>

Lingput was designed to production standards, with a focus on **scalability and clean architecture:**

<ul class="list-disc ml-6">
  <li>
    **Background processing:** BullMQ + Redis queue for long AI jobs
    (story/audio).
  </li>
  <li>
    **Clean backend:** Express.js with Dependency Injection & layered
    architecture.
  </li>
  <li>**Caching:** Redis caches stories and vocab for fast responses.</li>
  <li>**Authentication:** Secure cookies with access/refresh tokens.</li>
  <li>
    **Frontend:** Next.js with a custom React hook (handleJob) for job
    lifecycle.
  </li>
  <li>
    **Infrastructure:** Fully containerized with Docker & Compose, deployed via
    CapRover + DigitalOcean.
  </li>
</ul>
<br />
<span class="text-lg font-semibold">Tech Stack</span>

<ul class="list-disc ml-6">
  <li>**Frontend:** Next.js (App Router), React, Tailwind CSS</li>
  <li>**Backend:** Express.js, PostgreSQL (Prisma ORM)</li>
  <li>**Infra:** Docker, NGINX, GitHub Actions, CapRover, DigitalOcean</li>
  <li>**Workers:** BullMQ, Redis</li>
  <li>**Cloud Storage:** Supabase</li>
  <li>**AI:** OpenAI</li>
</ul>

<br />
<span class="text-lg font-semibold">Why I Built This</span>

I wanted to explore how AI could make language learning more **engaging and scalable**,
while also practicing real-world **system design, background processing, and DevOps.**
Building Lingput taught me a lot about **job queues, caching, containerization, CI/CD
pipelines, and clean architecture.**
